# IoT Botnet Detection System

Real-time network traffic monitoring for DDoS, Reconnaissance, and C&C attacks using Machine Learning.

## Features

-  Real-time network traffic analysis
-  ML-powered attack detection (Random Forest, 96% accuracy)
-  Live dashboard with interactive graphs
-  Detects DDoS, Reconnaissance, and C&C attacks
